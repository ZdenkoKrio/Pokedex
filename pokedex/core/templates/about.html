About this Django Pokédex

Minimal, cleanly structured app that integrates the PokeAPI, preloads data into a local cache, and renders fast SSR pages with rich filters & evolution chains.

✨ Highlights

  • ⚡
    DB-first performance
    Bulk sync do lokálnej DB (Pokémoni + evo chains), detail bez externých volaní.
  • 🧭
    Bohaté filtre
    Typ, ability, váha, generácia, legendárny/mytický…
  • 🧬
    Evolučné reťazce
    Predpočítané species_ids pre rýchle renderovanie a prekliky.
  • 📊
    Čistý kód
    Oddelené services/selectors, management commands, progres bar pre sync.

🧱 Tech Stack

Django Django ORM Requests PokeAPI ThreadPool DB Cache

📐 Architecture (short)

  • 📦
    services.api – HTTP + DB cache (ETag/TTL) & URL builder
  • 🔁
    services.sync* – bulk preload & resilient retries
  • 🧩
    selectors – čisté ORM dotazy pre view/template
  • 🗃️
    models.* – tenké cache tabuľky + taxonómie
